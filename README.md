# ATOM Messenger

## Introduction

Real time chat application built with Node.js, express, socket.io & vanilla javascript.

## Usage

### Installing the Application

```bash
cd web
npm install
cd ..
cp sampleENV .env
docker-compose build
```

### Run the Application

```bash
docker-compose up -d
```

### Close the Application

```bash
docker-compose down
```
